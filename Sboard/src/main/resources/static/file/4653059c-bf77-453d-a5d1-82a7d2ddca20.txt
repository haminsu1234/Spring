1) 로그인 시연
1.1 로그인 ( 관리자 + 일반유저) 
1.2 로그인 실패시
1.3 회원가입 (일반유저 + 관리자) 회원가입시 필수약관 전부동의시 등록화면으로 넘어가는지
1.4 등록화면 : 유효성검사 + 다안되면 입력하라는 문구 띄우는것
1.5 메인페이지에서 30분간 동작 없으면 자동 로그아웃 기능
1.6 로그인 안하고 관리자 / 고객센터 /상품구매/ 장바구니 진입시 로그인 필터 체크


-> 관리자로 로그인 해두기 


2) 관리자 상품등록(상품 테이블 확인) 시연
2.1 관리자 페이지 진입 
2.2 관리자 상품등록
2.3 관리자 상품등록시 상품 목록에서 보이는지 (제일 마지막 페이지 보여주기)

-> 관리자 페이지 끝나고 구매자로 로그인 
3) 상품목록(페이징, 정렬) 시연
3.1 아무 상품 카테고리 클릭
3.2 카테고리 전체 상품 목록 구현된것 보여주고 낮은 or 높은 가격순등등 정렬해서 페이징 처리 보여주기

4) 상품보기(상품 갯수, 바로 구매하기, 장바구니) 시연(ORDER)
4.1 상품보기 (상품목록, 메인 인덱스 페이지) 에서 들어가는것 보여주기 
4.2 보기에서 숫자 올라가고 내려가는것 보여주기 (갯수에따라 가격변동 하는것 보여주기)
4.3 갯수 임의로 선택해서 바로구매하기 보여주기( 선택창 나오는데 취소버튼시 안넘어가는것 보여주기)

5) 장바구니(삭제, 선택주문, 카트 테이블 확인) 시연
5.1 상품 장바구니에 담는것 보여드리기 만일 장바구니에 동일 상품이 있다면 동일 상품 
5.2 CART 테이블 보여드리고
5.3 선택시 결제 내용 변경되는것 보여드리고 (한번 선택하면 절대 풀지 않기)

6) 주문하기(포인트적용, 주문완료, 주문 테이블 확인) 시연(COMPLETE)

6.1 포인트 적용되는지 (조건 1 . 회원포인트가 5000이상인지 -> 조건2 사용할려는 포인트가 5000이상인지) 조건 보여드리고 가격변동에 적용되는지
6.2 결제방법은 기본으로 신용카드 결제로 선택 되어있다는것 설명드리고
6.3 우편번호 검색되는것 보여드리고
결제완료시 장바구니 상품일시 카트에서 삭제하고 오더테이블에 주문 정보 총합이 들어가고 오더아이템에는 해당총합정보의 상품 개별의 정보가 들어감
6.4주문 정보 가져와지는점 설명 ( 포인트 적용해서) 
6.5주문테이블(ORDER)테이블 보여드리기(다른조가 안보여주면 말로만 설명 ~데이터 들어갑니다)
6.6 COMPLETE 로 넘어가면 포인트 적립됨 MEMBER POINT 테이블에 포인트 적립내역 보여드리기(다른조가 테이블 오픈하면)
6.7 MEMBER 테이블에서 포인트 사용/적립 총합 적용된것 보여드리기(테이블오픈시)
